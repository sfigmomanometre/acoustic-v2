# Data klasörü
Bu klasörde test kayıtları ve kalibrasyon verileri saklanır.

## Alt Klasörler

- `recordings/`: Test ses kayıtları (WAV, HDF5 formatında)
- `calibration/`: Kalibrasyon verileri (mikrofon hassasiyeti, faz düzeltmeleri)

**Not**: Bu klasördeki dosyalar `.gitignore` ile izlenmez (çok büyük olabilir).
